To run our project:
1) open frontend and backend in sperate terminal windows
2) .env file for the backend should include 
  DB_CONNECTION= " "
  AWS_ACCESS_KEY=" "
  AWS_SECRET_KEY=" "
  AWS_BUCKET_NAME="data-sphere-app-s3"
  AWS_BUCKET_REGION="us-east-1"
  PORT=8080 
3) run "npm install node-modules" in both windows
4) run "npm start" on the backend window
5) run "npm start" on the frontend window
